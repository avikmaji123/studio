# **App Name**: CourseVerse

## Core Features:

- Secure User Authentication: Provides secure registration, login, and password management.
- Course Catalog & Detail: Enables browsing courses, viewing detailed descriptions, and enrolling.
- Lesson Player & Progress Tracking: Interactive lesson player with progress tracking and completion certificates.
- UPI Payment Processing: Manual UPI payment approval system, and point redemption for courses.
- Affiliate Program: Unique affiliate codes, commission tracking, and points-based rewards for referrals.
- Admin Command Center: Comprehensive admin panel for user, course, and payment management, and detailed analytics.
- Automated Fraud Detection Tool: When enabled, uses trust scoring, duplicate detection and screenshot reuse detection to prevent fraud. Offers action suggestions for the admin to override.

## Style Guidelines:

- Light theme: Background: #F6F7FB, Surface: #FFFFFF, Primary: #2563EB, Primary Hover: #1D4ED8, Accent: #F59E0B, Text Primary: #0F172A, Text Secondary: #6B7280, Border: #E5E7EB. This palette provides a clean, professional aesthetic appropriate to the educational context.
- Dark theme: Background: #020617, Surface: #0F172A, Primary: #38BDF8, Primary Hover: #0EA5E9, Accent: #FACC15, Text Primary: #E5E7EB, Text Secondary: #9CA3AF, Border: #1F2937. The dark theme offers a high-contrast, modern feel with visual appeal.
- Font pairing: 'Inter' (sans-serif) for body text, known for its neutral and readable design, combined with 'Playfair' (serif) for headlines to add a touch of elegance.
- Responsive design using CSS variables to ensure compatibility across devices (mobile, tablet, desktop).
- Use a consistent set of icons to represent course categories, user actions, and other key elements.
- Subtle transitions and animations to enhance user engagement, such as loading indicators and hover effects.
